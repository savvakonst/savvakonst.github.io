class Express_ex¶

this is the main interface to deal with express. Firstly you should use parseText, it makes primary syntax analysis. Than you are able to get map of link names by using getParameterLinkNamesMap. For next step of full semantic analysis Express_ex instance require input ParameterIfs instances via setParameters() to determine types of parameters (sync/async, data type, time type ,frequency is exists) and consequently types of all variables and outputs. After steps mentioned above it is possible to fetch output parameters by getOutputParameters(). Now it is possible to configure parameters, just edit received via getOutputParameters() instances of ParameterIfs. Next use genJit() to build and link; and last step is run();
